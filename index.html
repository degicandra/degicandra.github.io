ABOUT

Saya merupakan lulusan S1 Teknik Informatika di Universitas Nasional. Berpengalaman sebagai IT Staff selama lebih dari 4 tahun. Selain itu saya juga merangkap jabatan sebagai perbantuan Data Analyst. Selama bekerja saya biasa menangani troubleshooting hardware (PC, laptop, printer, VoIP, dll) maupun software (SIMRS/ERP), maintenance perangkat komputer dan penunjangnya, konfigurasi VoIP / PABX, installasi CCTV, installasi router / AP, assets management dan budgeting, web development (dengan PHP, CSS dan JavaScript), desain grafis (Photoshop/Figma) dan juga video editing (Adobe Premiere).
